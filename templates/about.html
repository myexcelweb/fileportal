‚ú® Key Features

  • Real-Time Sync: Files appear instantly on all devices using WebSocket technology
  • No Registration: Start sharing immediately without creating an account
  • Auto-Delete: Rooms and files are automatically deleted after 15 minutes
  • Multiple Files: Upload several files at once (up to 100MB total)
  • QR Codes: Easy mobile access with generated QR codes
  • Download All: ZIP download of all files in a room
  • Cross-Platform: Works on desktop, tablet, and mobile

üîß Technology Stack

Python Flask Socket.IO WebSockets Eventlet Gunicorn HTML5 CSS3 JavaScript QRCode.js

The application runs on Render with eventlet workers for real-time WebSocket support. All data is stored in memory and on disk temporarily, with automatic cleanup every minute.

üîí Privacy & Security

Your privacy is our priority. Here's how we protect your data:

  • Temporary Storage: Files exist only for 15 minutes
  • No Tracking: We don't track or store user information
  • Room Isolation: Each room is completely separate
  • Auto-Cleanup: Expired files are permanently deleted
  • No Database: No permanent storage of any data
